2. Quick Start - top

If you don't need to make any customization, you just can upload the entire file in the dist folder to your hosting or Facebook Web Hosting for Instant Games.

To create new quiz, you can follow the following steps.

1. Update quizzes.json file.

In this file, you can specify the slug and the title of the quiz.

{
    "quiz-slug": "This is quiz title"
}

2. Create a folder in quizzes folder based on your slug.

Your quiz folder should be renamed based on slug that you enter in quizzes.json file and should contain the following files.

  • config.json
    • In this file, you can specify how many images, to show or hide the Facebook player name or pic. 
      {
    "images": "Specify the number of result images.",
    "player": {
        "name": {
            "font": "bold 45px Arial",
            "color": "#fff|rgba(0,0,0,.5)",
            "position": {
                "x": 300,
                "y": 135
            },
            "align": "left|right|center",
            "show": true|false
        },
        "pic": {
            "width": 200,
            "height": 200,
            "position": {
                "x": 77,
                "y": 283
            },
            "show": true|false
        }
    }
}
  • thumbnail.png - The thumbnail of the quiz. Will be displayed on home page.
  • List of images - Include all quiz result images in PNG format and should be renamed in numerical, starting from 1.

3. Generate production files

To generate production files or use the development server, you need to have Node.js installed. Click here to install Node.js.

Then, run npm install to install required package files.

To generate production files, follow this steps:

  1. Open terminal or command prompt.
  2. Type npm run build to generate production files.
  3. Done, your production folder is dist folder.

To use the development server, follow this steps:

  1. Open terminal or command prompt.
  2. Type npm run serve to start the development server.
  3. You may see the app is running through development server at http://localhost:8080/#/.

4. You're ready to go.

Now, your quiz app are ready to go. You can upload dist folder to your hosting or use Facebook Web Hosting to host your quiz app.

3. Customization - top

If you need to make customization such as title, styling, etc, you should use development server that will help you to create a local server where you can see the customization you have made on your browser.

Refer to Quick Start section on how to use the development server.

Update App Title

Open .env file on root directory and update VUE_APP_NAME variable to your new app title.

Include Facebook Audience Network Ad

Open .env file on root directory and update VUE_APP_FAN_AD_ID variable to Facebook Audience Network Ad Placement ID and update VUE_APP_FAN_AD_FREQUENCY variable to the frequency of showing ad to user (default is 3).

Customize Styling

You can customize to your own styling in src/assets/css/app.css.

Javascript

If you wish to include any script, you can put in src/assets/js/app.js.

If you have done make customization, open terminal or command prompt and type “npm run production” to generate updated files.
